## .env 
```
{
  "issuer": jwt issuer, could be your application name,
  "db_url": mopngodb database url,
  "db_user": mopngodb database username,
  "db_pass": mopngodb database password,
  "private_key": single line rsa private key, use \n for line breaks,
  "public_key": single line rsa pem public key, use \n for line breaks
}

```
